Resource Methods Duality Documentation
The Resource type exposes the following members.
Methods
  NameDescription
Clone
Creates a deep copy of this Resource.
CopyTo
Deep-copies this Resource to the specified target Resource. The target Resource's Type must match this Resource's Type.
Dispose
Disposes the Resource.
DisposeLater
Schedules this Resource for disposal. It is guaranteed to be executed until the next update cycle starts.
Finalize (Overrides Object Finalize .)
GetContentRef
Creates a ContentRef T  referring to this Resource.
GetFileExtByType
Returns the Resource file extension for a specific Resource Type.
GetResourceFiles
Returns all Resource files that are located in the specified directory. This doesn't affect any actual content- or load states.
GetTypeByFileName
Returns the Resource Type that is associated with the specified file, based on its extension.
IsResourceFile
Determines whether or not the specified path points to a Duality Resource file.
LoadResource T (String, Action T )
Loads the Resource that is located at the specified path. You usually don't need this method. Consider requesting the Resource from the ContentProvider instead.
LoadResource T (Stream, String, Action T )
Loads the Resource from the specified Stream. You usually don't need this method. Consider requesting the Resource from the ContentProvider instead.
NonSerializedResourceBlocker
A FieldBlocker to prevent fields flagged with a NonSerializedResourceAttribute from being serialized.
OnCopyTo
This method Performs the CopyTo(Resource) operation for custom Resource Types. It uses reflection to copy each field that is declared inside a Duality plugin automatically. However, you may override this method to specify your own behaviour or simply speed things up a bit by not using Reflection.
OnDisposing
Called when beginning to dispose the Resource.
OnLoaded
Called when this Resource has just been loaded.
OnSaved
Called when this Resource has just been saved.
OnSaving
Called when this Resource is now beginning to be saved.
Save(Stream)
Saves the Resource to the specified stream.
Save(String)
Saves the Resource to the specified path. If it has been generated at runtime, i.e. has not been loaded from file before, this will set the Resources Path Property.
