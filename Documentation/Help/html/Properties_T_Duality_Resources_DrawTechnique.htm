DrawTechnique Properties Duality Documentation
The DrawTechnique type exposes the following members.
Properties
  NameDescription
Add
Renders additive geometry. Ideal for glow effects.
Alpha
Renders geometry and using the alpha channel. However, for stencil-sharp alpha edges, Mask might be sufficient and is a lot faster. Consider using it.
Blending
[GET / SET] Specifies how incoming color values interact with the existing background color.
Disposed
[GET] Returns whether the Resource has been disposed. Disposed Resources are not to be used and are treated the same as a null value by most methods.
(Inherited from Resource.)
FullName
[GET] The full name of the Resource, including its path but not its file extension
(Inherited from Resource.)
Invert
Renders geometry inverting the background color.
IsDefaultContent
[GET] Returns whether the Resource is part of Duality's embedded default content.
(Inherited from Resource.)
IsRuntimeResource
[GET] Returns whether the Resource has been generated at runtime and cannot be retrieved via content path.
(Inherited from Resource.)
Light
Renders geometry adding incoming color values weighted based on the existing background. Can be used for lighting effects.
Mask
Renders alpha-masked solid geometry. This is the recommended DrawTechnique for regular sprite rendering. If multisampling is available, it is utilized to smooth masked edges.
MaskUseAlphaToCoverage
[GET] Returns whether the masked DrawTechniques utilize OpenGL Alpha-to-Coverage to smooth alpha-edges in masked rendering.
Multiply
Renders geometry multiplying the existing background with incoming color values. Can be used for shadowing effects.
Name
[GET] The name of the Resource.
(Inherited from Resource.)
NeedsPreparation
[GET] Returns whether this DrawTechnique requires any rendering preparation. This is false for all standard DrawTechniques, but may return true when deriving custom DrawTechniques.
NeedsPreprocess
[GET] Returns whether this DrawTechnique requires vertex preprocessing. This is false for all standard DrawTechniques, but may return true when deriving custom DrawTechniques.
NeedsZSort
[GET] Returns whether this DrawTechnique requires z sorting. It is derived from its Blending.
Path
[GET] The path where this Resource has been originally loaded from or was first saved to. It is also the path under which this Resource is registered at the ContentProvider.
(Inherited from Resource.)
Picking
Renders geometry for a picking operation. This isn't used for regular rendering.
PreferredVertexFormat
[GET / SET] The vertex format that is preferred by this DrawTechnique. If there is no specific preference, VertexType_Unknown is returned.
Shader
[GET / SET] The ShaderProgram that is used for rendering.
SmoothAnim_Add
Renders SmoothAnim additive geometry. Ideal for glow effects.
SmoothAnim_Alpha
Renders SmoothAnim geometry and using the alpha channel. However, for stencil-sharp alpha edges, Mask might be sufficient and is a lot faster. Consider using it.
SmoothAnim_Invert
Renders SmoothAnim geometry inverting the background color.
SmoothAnim_Light
Renders SmoothAnim geometry adding incoming color values weighted based on the existing background. Can be used for lighting effects.
SmoothAnim_Mask
Renders SmoothAnim alpha-masked solid geometry. This is the recommended DrawTechnique for regular sprite rendering. If multisampling is available, it is utilized to smooth masked edges.
SmoothAnim_Multiply
Renders SmoothAnim geometry multiplying the existing background with incoming color values. Can be used for shadowing effects.
SmoothAnim_Solid
Renders SmoothAnim solid geometry without utilizing the alpha channel. This is the fastest default DrawTechnique.
Solid
Renders solid geometry without utilizing the alpha channel. This is the fastest default DrawTechnique.
SourcePath
[GET / SET] The path of the file from which the Resource has been originally imported or initialized. Setting this does not affect the Resource in any way.
(Inherited from Resource.)
