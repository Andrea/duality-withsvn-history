ContentProvider Members Duality Documentation

The ContentProvider type exposes the following members.
Methods
  NameDescription
ClearContent
Clears all non-default content.
GetAllDefaultContent
Returns a list of all available embedded default content.
GetAvailContent(Type)
Returns a list of all available content matching the specified Type
GetAvailContent T  
Returns a list of all available content matching the specified Type
InitDefaultContent
Initializes Dualitys embedded default content.
IsContentRegistered
Returns whether or not there is any content currently registered under the specified path key.
RegisterContent
Registers a Resource and maps it to the specified path key.
RenameContent
Changes the path key under which a specific Resource can be found. If the target path is registered already, it will be replaced. If the source path isn't registered / is unknown, the operation will fail.
RenameContentTree
Changes the path key under which a set of Resource can be found, i.e. renames all path keys located inside the specified directory.
RequestContent(String)
Requests a Resource.

Requests a Resource.
UnregisterAllContent(Type, Boolean)
Unregisters all content of the specified Type or subclassed Types.
UnregisterAllContent T (Boolean)
Unregisters all content of the specified Type or subclassed Types.
UnregisterContent
Unregisters content that has been registered using the specified path key.
UnregisterContentTree
Unregisters all content that has been registered using paths contained within the specified directory.
