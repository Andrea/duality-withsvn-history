Scene Properties Duality Documentation
The Scene type exposes the following members.
Properties
  NameDescription
ActiveObjects
[GET] Enumerates all registered objects that are currently active.
ActiveRootObjects
[GET] Enumerates all RootObjects that are currently active.
AllObjects
[GET] Enumerates all registered objects.
Cameras
[GET] Enumerates the Scenes Camera objects.
Current
[GET / SET] The Scene that is currently active i.e. updated and rendered. This is never null. You may assign null in order to leave the current Scene and enter en empty dummy Scene.
CurrentPath
[GET] The Resource file path of the current Scene.
Disposed
[GET] Returns whether the Resource has been disposed. Disposed Resources are not to be used and are treated the same as a null value by most methods.
(Inherited from Resource.)
FullName
[GET] The full name of the Resource, including its path but not its file extension
(Inherited from Resource.)
GlobalGravity
[GET / SET] Global gravity force that is applied to all objects that obey the laws of physics.
IsCurrent
[GET] Returns whether this Scene is Current.
IsDefaultContent
[GET] Returns whether the Resource is part of Duality's embedded default content.
(Inherited from Resource.)
IsEmpty
[GET] Returns whether this Scene is completely empty.
IsRuntimeResource
[GET] Returns whether the Resource has been generated at runtime and cannot be retrieved via content path.
(Inherited from Resource.)
Name
[GET] The name of the Resource.
(Inherited from Resource.)
Path
[GET] The path where this Resource has been originally loaded from or was first saved to. It is also the path under which this Resource is registered at the ContentProvider.
(Inherited from Resource.)
PhysicsAlpha
[GET] When using fixed-timestep physics, the alpha value [0.0 - 1.0] indicates how complete the next step is. This is used for linear interpolation inbetween fixed physics steps.
PhysicsFixedTime
[GET] Is fixed-timestep physics calculation currently active?
PhysicsWorld
[GET] Returns the current physics world.
Renderers
[GET] Enumerates the Scenes Renderer objects.
RootObjects
[GET] Enumerates all root GameObjects, i.e. all GameObjects without a parent object.
SourcePath
[GET / SET] The path of the file from which the Resource has been originally imported or initialized. Setting this does not affect the Resource in any way.
(Inherited from Resource.)
