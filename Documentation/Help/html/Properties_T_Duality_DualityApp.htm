DualityApp Properties Duality Documentation
The DualityApp type exposes the following members.
Properties
  NameDescription
AppData
[GET / SET] Provides access to Duality's current application data. This is never null. Any kind of data change event is fired as soon as you re-assign this property. Be sure to do that after changing its data.
AppDataPath
[GET] Returns the path where this DualityApp's application data is located at.
AvailableModes
[GET] Enumerates all available GraphicsModes.
DefaultMode
[GET] Returns the GraphicsMode that Duality intends to use by default.
DisposedPlugins
[GET] Enumerates all plugin assemblies that have been loaded before, but have been discarded due to a runtime plugin reload operation. This is usually only the case when being executed from withing the editor or manually triggering a plugin reload. However, this is normally unnecessary.
ExecContext
[GET] Returns the DualityApp ExecutionContext in which this DualityApp is currently running.
ExecEnvironment
[GET] Returns the DualityApp ExecutionEnvironment in which this DualityApp is currently running.
Keyboard
[GET] Provides access to keyboard user input
LoadedPlugins
[GET] Enumerates all currently loaded plugins.
MetaData
[GET] Provides access to Duality's current meta data. This is never null.
MetaDataPath
[GET] Returns the path where this DualityApp's meta data is located at.
Mouse
[GET] Provides access to mouse user input.
Sound
[GET] Provides access to the main SoundDevice.
TargetMode
[GET / SET] The GraphicsMode in which rendering takes place. Setting this will not actually change Duality's state - this is a pure "for your information" property.
TargetResolution
[GET / SET] The size of the current rendering surface (full screen, a single window, etc.) in pixels. Setting this will not actually change Duality's state - this is a pure "for your information" property.
UserData
[GET / SET] Provides access to Duality's current user data. This is never null. Any kind of data change event is fired as soon as you re-assign this property. Be sure to do that after changing its data.
UserDataPath
[GET] Returns the path where this DualityApp's user data is located at.
