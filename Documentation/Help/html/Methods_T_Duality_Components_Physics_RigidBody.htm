ApplyLocalForce(Vector2)
Applies a Transform-local force to the objects mass center. You don't need to apply TimeMult here, the physics simulation takes care of this.
ApplyLocalForce(Single)
Applies a Transform-local angular force to the object. You don't need to apply TimeMult here, the physics simulation takes care of this.
ApplyLocalForce(Vector2, Vector2)
Applies a Transform-local force to the specified local point. You don't need to apply TimeMult here, the physics simulation takes care of this.
ApplyLocalImpulse(Vector2)
Applies a Transform-local impulse to the objects mass center. You don't usually need to apply TimeMult here because it is inteded to be a one-time force impact.
ApplyLocalImpulse(Single)
Applies a Transform-local angular impulse to the object. You don't usually need to apply TimeMult here because it is inteded to be a one-time force impact.
ApplyLocalImpulse(Vector2, Vector2)
Applies a Transform-local impulse to the specified point. You don't usually need to apply TimeMult here because it is inteded to be a one-time force impact.
ApplyWorldForce(Vector2)
Applies a world force to the objects mass center. You don't need to apply TimeMult here, the physics simulation takes care of this.
ApplyWorldForce(Vector2, Vector2)
Applies a world force to the specified world point. You don't need to apply TimeMult here, the physics simulation takes care of this.
ApplyWorldImpulse(Vector2)
Applies a world impulse to the objects mass center. You don't usually need to apply TimeMult here because it is inteded to be a one-time force impact.
ApplyWorldImpulse(Vector2, Vector2)
Applies a world impulse to the specified point. You don't usually need to apply TimeMult here because it is inteded to be a one-time force impact.
