Camera Properties Duality Documentation
The Camera type exposes the following members.
Properties
  NameDescription
Active
[GET / SET] Whether or not the Component is currently active. To return true, both the Component itsself and its parent GameObject need to be active.
(Inherited from Component.)
ActiveSingle
[GET / SET] Whether or not the Component is currently active. Unlike Active, this property ignores parent activation states and depends only on this single Component. The scene graph and other Duality instances usually check Active, not ActiveSingle.
(Inherited from Component.)
ClearColor
[GET / SET] The background color of the rendered image.
Disposed
[GET] Returns whether this Component has been disposed. Disposed Components are not to be used and should be treated specifically or as null references by your code.
(Inherited from Component.)
DrawDevice
[GET] The drawing device which this Camera uses for rendering.
FarZ
[GET / SET] The highest Z value that can be displayed by the device.
FocusDist
[GET / SET] Reference distance for calculating the view projection. When using Parallax, an object this far away from the Camera will always appear in its original size and without offset.
GameObj
[GET / SET] The GameObject to which this Component belongs.
(Inherited from Component.)
NearZ
[GET / SET] The lowest Z value that can be displayed by the device.
Passes
[GET / SET] A set of passes that describes the Cameras rendering process. Is never null nor empty.
Perspective
[GET / SET] Specified the perspective effect that is applied when rendering the world.
SceneOrthoAbs
[GET] A Rect describing the Cameras absolute ortho value.
SceneTargetSize
[GET] The drawing devices target size for rendering the Scene.
SceneViewportAbs
[GET] A Rect describing the Cameras absolute viewport.
ViewBoundingRadius
[GET] The Cameras view space bounding circle radius.
VisibilityMask
[GET / SET] A bitmask flagging all visibility groups that are considered visible to this drawing device.
