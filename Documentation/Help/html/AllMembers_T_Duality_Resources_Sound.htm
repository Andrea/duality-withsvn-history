Properties
  NameDescription
Beep
[GET] A simple beep Sound.
Data
[GET / SET] The AudioData that is parameterized by this Sound.
Disposed
[GET] Returns whether the Resource has been disposed. Disposed Resources are not to be used and are treated the same as a null value by most methods.
(Inherited from Resource.)
DroneLoop
[GET] A drone loop Sound. Since this is a stereo Sound, it will always be played 2D.
FadeOutAt
[GET / SET] Play time in seconds at which SoundInstances of this Sound automatically fade out.
FadeOutTime
[GET / SET] If FadeOutAt has been triggered, this is the fade out time in seconds that is used.
FullName
[GET] The full name of the Resource, including its path but not its file extension
(Inherited from Resource.)
IsDefaultContent
[GET] Returns whether the Resource is part of Duality's embedded default content.
(Inherited from Resource.)
IsRuntimeResource
[GET] Returns whether the Resource has been generated at runtime and cannot be retrieved via content path.
(Inherited from Resource.)
IsStreamed
[GET] Returns whether the Sound will be streamed when playing.
LogoJingle
[GET] A logo jingle Sound. Since this is a stereo Sound, it will always be played 2D.
MaxDist
[GET / SET] The distance at which the sound is played at zero volume.
MaxDistFactor
[GET / SET] The distance at which the sound is played at zero volume. Note that the value is a factor to DefaultMaxDist.
MaxInstances
[GET / SET] Maximum number of SoundInstances of this Sound that can play simultaneously. If exceeded, any new instances of it are discarded.
MinDist
[GET / SET] The distance at which the sound is played at full volume. Getting nearer to the source won't increase the volume anymore.
MinDistFactor
[GET / SET] The distance at which the sound is played at full volume. Getting nearer to the source won't increase the volume anymore. Note that the value is a factor to DefaultMinDist.
Name
[GET] The name of the Resource.
(Inherited from Resource.)
Path
[GET] The path where this Resource has been originally loaded from or was first saved to. It is also the path under which this Resource is registered at the ContentProvider.
(Inherited from Resource.)
PitchFactor
[GET / SET] A pitch factor that is applied when playing this sound.
SourcePath
[GET / SET] The path of the file from which the Resource has been originally imported or initialized. Setting this does not affect the Resource in any way.
(Inherited from Resource.)
Type
[GET / SET] The category to which this Sound belongs.
VolumeFactor
[GET / SET] A volume factor that is applied when playing this sound.
