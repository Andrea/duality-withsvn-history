Pixmap Layer Methods Duality Documentation
The Pixmap Layer type exposes the following members.
Methods
  NameDescription
ClearPixelData
Discards all pixel data in this Layer.
Clone
Clones the pixel data layer and returns the new instance.
CloneCrop
Crops the Layer, removing transparent / empty border areas.
CloneRescale
Rescales the Layer, stretching it to the specified size.
CloneResize
Resizes the Layers boundaries.
CloneSubImage(Int32, Int32, Int32, Int32)
Extracts a rectangular region of this Layer. If the extracted region is bigger than the original Layer, all new space is filled with a background color.
CloneSubImage(Int32, Int32, Int32, Int32, ColorRgba)
Extracts a rectangular region of this Layer. If the extracted region is bigger than the original Layer, all new space is filled with a background color.
ColorTransparentPixels 
Sets the color of all transparent pixels based on the non-transparent color values next to them. This does not affect any alpha values but prepares the Layer for correct filtering once uploaded to Texture.
ColorTransparentPixels(ColorRgba)
Sets the color of all transparent pixels to the specified color.
CopyTo
Copies all data contained in this pixel data layer to a target layer.
Crop
Crops the Layer, removing transparent / empty border areas.
DrawOnto(Pixmap Layer, BlendMode, Int32, Int32, Int32, Int32, Int32, Int32)
Performs a drawing operation from this Layer to a target layer.
DrawOnto(Pixmap Layer, BlendMode, Int32, Int32, Int32, Int32, Int32, Int32, ColorRgba)
Performs a drawing operation from this Layer to a target layer.
FromBitmap
Sets this Layers pixel data to the one contained in the specified Bitmap
GetAverageColor
Determines the average color of a Layer.
GetPixelDataByteRgba
Gets the Layers pixel data in bytewise Rgba format. (Four elements per pixel)
GetPixelDataIntArgb
Gets the Layers pixel data in the integer Argb format. (One element per pixel)
GetPixelDataRgba
Gets the Layers pixel data in the ColorRgba format. Note that this data is a clone and thus modifying it won't affect the Layer it has been retrieved from.
LoadPixelData
Loads the pixel data in this layer from the specified file.
OpaqueBounds
Measures the bounding rectangle of the Layers opaque pixels.
Rescale
Rescales the Layer, stretching it to the specified size.
Resize
Resizes the Layers boundaries.
SavePixelData
Saves the pixel data contained in this layer to the specified file.
SetPixelDataArgb
Sets the Layers pixel data in the integer Argb format. (One element per pixel)
SetPixelDataRgba( Byte , Int32, Int32)
Sets the Layers pixel data in bytewise Rgba format. (Four elements per pixel)
SetPixelDataRgba( ColorRgba , Int32, Int32)
Sets the layers pixel data in the ColorRgba format. Note that the specified data will be copied and thus modifying it outside won't affect the Layer it has been inserted into.
SubImage(Int32, Int32, Int32, Int32)
Extracts a rectangular region of this Layer. If the extracted region is bigger than the original Layer, all new space is filled with a background color.
SubImage(Int32, Int32, Int32, Int32, ColorRgba)
Extracts a rectangular region of this Layer. If the extracted region is bigger than the original Layer, all new space is filled with a background color.
ToBitmap
Creates a Bitmap out of this Layer.
