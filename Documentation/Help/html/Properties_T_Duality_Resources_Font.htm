Font Properties Duality Documentation
The Font type exposes the following members.
Properties
  NameDescription
Ascent
[GET] The Fonts ascent value.
BaseLine
[GET] The Fonts base line height.
BodyAscent
[GET] The Fonts body ascent value.
CharSpacing
[GET / SET] Additional spacing between each character. This is usually one tenth of the Fonts Size.
CustomFamilyData
[GET] Returns a chunk of memory that contains this Fonts custom family data.
Descent
[GET] The Fonts descent value.
Disposed
[GET] Returns whether the Resource has been disposed. Disposed Resources are not to be used and are treated the same as a null value by most methods.
(Inherited from Resource.)
Family
[GET / SET] The name of the font family that is used.
Filtering
[GET / SET] Whether or not this Font uses texture filtering. Note that if texture filtering is enabled, fonts may appear slightly blurred due to full screen antialiazing / multisampling.
FullName
[GET] The full name of the Resource, including its path but not its file extension
(Inherited from Resource.)
GenericMonospace10
A generic monospace Font (Size 10) that has been loaded from your systems font library. This is usually "Courier New".
GenericMonospace8
A generic monospace Font (Size 8) that has been loaded from your systems font library. This is usually "Courier New".
GenericSansSerif12
A generic sans-serif Font (Size 12) that has been loaded from your systems font library. This is usually "Arial".
GenericSerif12
A generic serif Font (Size 12) that has been loaded from your systems font library. This is usually "Times New Roman".
GlyphBgColor
[GET / SET] When pre-rasterizing the necessary glyphs, their background has this color.
GlyphColor
[GET / SET] When pre-rasterizing the necessary glyphs, they are rendered in this color.
GlyphRenderHint
[GET / SET] Configures the internal glyph rasterizer.
Height
[GET] The Fonts height.
IsDefaultContent
[GET] Returns whether the Resource is part of Duality's embedded default content.
(Inherited from Resource.)
IsRuntimeResource
[GET] Returns whether the Resource has been generated at runtime and cannot be retrieved via content path.
(Inherited from Resource.)
Kerning
[GET / SET] Whether this Font uses kerning, a technique where characters are moved closer together based on their actual shape, which usually looks much nicer. It has no visual effect when active at the same time with MonoSpace, however kerning sample data will be available on glyphs.
LineHeightFactor
[GET / SET] A factor for the Fonts Height value that affects line spacings but not actual glyph sizes.
LineSpacing
[GET] The y offset in pixels between two lines.
Material
[GET] The Material to use when rendering text of this Font.
MonoSpace
[GET / SET] Whether this is considered a monospace Font. If true, each character occupies exactly the same space.
Name
[GET] The name of the Resource.
(Inherited from Resource.)
NeedsReload
[GET] Returns whether this Font needs a reload in order to apply changes that have been made to its Properties.
Path
[GET] The path where this Resource has been originally loaded from or was first saved to. It is also the path under which this Resource is registered at the ContentProvider.
(Inherited from Resource.)
Size
[GET / SET] The size of the Font.
SourcePath
[GET / SET] The path of the file from which the Resource has been originally imported or initialized. Setting this does not affect the Resource in any way.
(Inherited from Resource.)
Style
[GET / SET] The style of the font.
