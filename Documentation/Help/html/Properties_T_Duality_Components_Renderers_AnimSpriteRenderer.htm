AnimSpriteRenderer Properties Duality Documentation
The AnimSpriteRenderer type exposes the following members.
Properties
  NameDescription
Active
[GET / SET] Whether or not the Component is currently active. To return true, both the Component itsself and its parent GameObject need to be active.
(Inherited from Component.)
ActiveSingle
[GET / SET] Whether or not the Component is currently active. Unlike Active, this property ignores parent activation states and depends only on this single Component. The scene graph and other Duality instances usually check Active, not ActiveSingle.
(Inherited from Component.)
AnimDuration
[GET / SET] The time a single animation cycle needs to complete, in seconds.
AnimFirstFrame
[GET / SET] The index of the first frame to display. Ignored if CustomFrameSequence is set.
AnimFrameCount
[GET / SET] The number of continous frames to use for the animation. Ignored if CustomFrameSequence is set.
AnimLoopMode
[GET / SET] The animations loop behaviour.
AnimTime
[GET / SET] The animations current play time, i.e. the current state of the animation.
BoundRadius (Inherited from SpriteRenderer.)
ColorTint
[GET / SET] A color by which the sprite is tinted.
(Inherited from SpriteRenderer.)
CurrentFrame
[GET] The currently visible animation frames index.
CurrentFrameProgress
[GET] The current animation frames progress where zero means "just entered the current frame" and one means "about to leave the current frame". This value is also used for smooth animation blending.
CustomFrameSequence
[GET / SET] A custom sequence of frame indices that will be used instead of the default range specified by AnimFirstFrame and AnimFrameCount. Unused if set to null.
CustomMaterial
[GET / SET] A custom, local BatchInfo overriding the SharedMaterial, allowing this sprite to look unique without having to create its own Material Resource. However, this feature should be used with caution: Performance is better using shared Materials.
(Inherited from SpriteRenderer.)
Disposed
[GET] Returns whether this Component has been disposed. Disposed Components are not to be used and should be treated specifically or as null references by your code.
(Inherited from Component.)
GameObj
[GET / SET] The GameObject to which this Component belongs.
(Inherited from Component.)
IsAnimationRunning
[GET] Whether the animation is currently running, i.e. if there is anything animated right now.
NextFrame
[GET] The next visible animation frames index.
Offset
[GET / SET] A virtual Z offset that affects the order in which objects are drawn. If you want to assure an object is drawn after another one, just assign a higher Offset value to the background object.
(Inherited from SpriteRenderer.)
Rect
[GET / SET] The rectangular area the sprite occupies. Relative to the GameObject.
(Inherited from SpriteRenderer.)
RectMode
[GET / SET] Specifies how the sprites uv-Coordinates are calculated.
(Inherited from SpriteRenderer.)
SharedMaterial
[GET / SET] The Material that is used for rendering the sprite.
(Inherited from SpriteRenderer.)
VertexZOffset
[GET] The internal Z-Offset added to the renderers vertices based on its Offset value.
(Inherited from SpriteRenderer.)
VisibilityGroup
[GET / SET] A bitmask that informs about the set of visibility groups to which this Renderer belongs. Usually, a Renderer is considered visible to a Camera if they share at least one mutual visibility group.
(Inherited from Renderer.)
