Texture Properties Duality Documentation
The Texture type exposes the following members.
Properties
  NameDescription
AnimCols
[GET] The number of animation frame cols in this Texture
AnimFrames
[GET] Total number of animation frames in this Texture
AnimRows
[GET] The number of animation frame rows in this Texture
BasePixmap
[GET / SET] Reference to a Pixmap that contains the pixel data that is or has been uploaded to the Texture
BoundTex
[GET] All Textures that are currently bound
BoundTexPrimary
[GET] The currently bound primary Texture.
BoundTexQuartary
[GET] The currently bound quartary Texture
BoundTexSecondary
[GET] The currently bound secondary Texture
BoundTexTertiary
[GET] The currently bound tertiary Texture
Disposed
[GET] Returns whether the Resource has been disposed. Disposed Resources are not to be used and are treated the same as a null value by most methods.
(Inherited from Resource.)
DualityLogo256
[GET] A Texture showing the Duality logo.
DualityLogoB256
[GET] A Texture showing the Duality logo without the text on it.
FilterMag
[GET / SET] The Textures magnifying filter
FilterMin
[GET / SET] The Textures minifying filter
FullName
[GET] The full name of the Resource, including its path but not its file extension
(Inherited from Resource.)
HasMipmaps
[GET] Returns whether or not the texture uses mipmaps.
IsDefaultContent
[GET] Returns whether the Resource is part of Duality's embedded default content.
(Inherited from Resource.)
IsRuntimeResource
[GET] Returns whether the Resource has been generated at runtime and cannot be retrieved via content path.
(Inherited from Resource.)
Name
[GET] The name of the Resource.
(Inherited from Resource.)
NeedsReload
Indicates that the textures parameters have been changed in a way that will make it necessary to reload its data before using it next time.
Path
[GET] The path where this Resource has been originally loaded from or was first saved to. It is also the path under which this Resource is registered at the ContentProvider.
(Inherited from Resource.)
PixelFormat
[GET / SET] The Textures pixel format
PixelHeight
[GET] The Textures original pixel height
PixelWidth
[GET] The Textures original pixel width
Size
[GET / SET] The Textures size. Readonly, when created from a BasePixmap.
SizeRelative
[GET / SET] If true, Size will be multiplied with TargetResolution to determine the actual texture size. This essentially makes the Textures size relative to the current screen resolution.
SourcePath
[GET / SET] The path of the file from which the Resource has been originally imported or initialized. Setting this does not affect the Resource in any way.
(Inherited from Resource.)
TexelHeight
[GET] The Textures internal texel height
TexelWidth
[GET] The Textures internal texel width
TexSizeMode
[GET / SET] Handles how the Textures base Pixmap is adjusted in order to fit GPU texture size requirements (Power of Two dimensions)
UVRatio
[GET] UV (Texture) coordinates for the Textures lower right
White
[GET] A plain white 1x1 Texture. Can be used as a dummy.
WrapX
[GET / SET] The Textures horizontal wrap mode
WrapY
[GET / SET] The Textures vertical wrap mode
