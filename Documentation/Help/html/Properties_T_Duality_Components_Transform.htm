Transform Properties Duality Documentation
The Transform type exposes the following members.
Properties
  NameDescription
Active
[GET / SET] Whether or not the Component is currently active. To return true, both the Component itsself and its parent GameObject need to be active.
(Inherited from Component.)
ActiveSingle
[GET / SET] Whether or not the Component is currently active. Unlike Active, this property ignores parent activation states and depends only on this single Component. The scene graph and other Duality instances usually check Active, not ActiveSingle.
(Inherited from Component.)
Angle
[GET / SET] The objects angle / rotation, in radians.
AngleVel
[GET] The objects angle / rotation velocity, in radians.
DeriveAngle
[GET / SET] If false, this objects rotation values aren't relative to its parent. However, its position, velocity, etc. still depend on parent rotation.
Disposed
[GET] Returns whether this Component has been disposed. Disposed Components are not to be used and should be treated specifically or as null references by your code.
(Inherited from Component.)
Forward
[GET] The objects forward vector.
GameObj
[GET / SET] The GameObject to which this Component belongs.
(Inherited from Component.)
IgnoreParent
[GET / SET] Specifies whether the Transform component should ignore its parent transform.
Pos
[GET / SET] The objects position.
RelativeAngle
[GET / SET] The objects angle / rotation relative to its parent object, in radians.
RelativeAngleVel
[GET / SET] The objects angle / rotation velocity relative to its parent object, in radians.
RelativeForward
[GET] The objects forward vector, relative to its parent object.
RelativePos
[GET / SET] The objects position relative to its parent object.
RelativeRight
[GET] The objects right (directional) vector, relative to its parent object.
RelativeScale
[GET / SET] The objects scale relative to its parent object.
RelativeVel
[GET / SET] The objects velocity relative to its parent object.
Right
[GET] The objects right (directional) vector.
Scale
[GET / SET] The objects scale.
Vel
[GET] The objects velocity.
