Properties
  NameDescription
DisplayedText
[GET] The text that is actually displayed.
Elements
[GET] The formatted text elements that have been generated analyzing the incoming source text.
FlowAreas
[GET / SET] A set of flow areas to be considered in word wrap. If you modify this value without re-assigning it, be sure to call UpdateVertexCache .
Fonts
[GET / SET] A set of Fonts that is available in the text. If you modify this value without re-assigning it, be sure to call UpdateVertexCache .
Icons
[GET / SET] A set of icons that is available in the text. If you modify this value without re-assigning it, be sure to call UpdateVertexCache .
MaxHeight
[GET / SET] The maximum height of the displayed text block. Zero deactivates maximum height.
MaxWidth
[GET / SET] The maximum width of the displayed text block. Zero deactivates maximum width.
SourceText
[GET / SET] The source text that is used, containing all format strings as well as the displayed text.
WordWrap
[GET / SET] The word wrapping behaviour to use.
