Pixmap Properties Duality Documentation
The Pixmap type exposes the following members.
Properties
  NameDescription
AnimCols
[GET / SET] Information about different animation frames contained in this Pixmap. Setting this will lead to an auto-generated atlas map according to the animation.
AnimFrames
[GET] Total number of animation frames in this Pixmap
AnimRows
[GET / SET] Information about different animation frames contained in this Pixmap. Setting this will lead to an auto-generated atlas map according to the animation.
Atlas
[GET / SET] The Pixmaps atlas array, distinguishing different areas in pixel coordinates
Disposed
[GET] Returns whether the Resource has been disposed. Disposed Resources are not to be used and are treated the same as a null value by most methods.
(Inherited from Resource.)
DualityLogo256
[GET] A Pixmap showing the Duality logo.
DualityLogoB256
[GET] A Pixmap showing the Duality logo without the text on it.
FullName
[GET] The full name of the Resource, including its path but not its file extension
(Inherited from Resource.)
Height
[GET] The Height of the actual pixel data.
IsDefaultContent
[GET] Returns whether the Resource is part of Duality's embedded default content.
(Inherited from Resource.)
IsRuntimeResource
[GET] Returns whether the Resource has been generated at runtime and cannot be retrieved via content path.
(Inherited from Resource.)
MainLayer
[GET / SET] The main pixel data Pixmap Layer.
Name
[GET] The name of the Resource.
(Inherited from Resource.)
Path
[GET] The path where this Resource has been originally loaded from or was first saved to. It is also the path under which this Resource is registered at the ContentProvider.
(Inherited from Resource.)
PixelData
[GET / SET] A list of pixel data Layers.
SourcePath
[GET / SET] The path of the file from which the Resource has been originally imported or initialized. Setting this does not affect the Resource in any way.
(Inherited from Resource.)
White
[GET] A plain white 1x1 Pixmap. Can be used as a dummy.
Width
[GET] The Width of the actual pixel data.
