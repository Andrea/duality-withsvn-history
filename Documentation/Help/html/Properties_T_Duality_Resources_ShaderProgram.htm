ShaderProgram Properties Duality Documentation
The ShaderProgram type exposes the following members.
Properties
  NameDescription
AttribCount
[GET] Returns the number of vertex attributes that have been declared.
BoundProgram
[GET] Returns the currently bound ShaderProgram.
Compiled
[GET] Returns whether this ShaderProgram has been compiled.
Disposed
[GET] Returns whether the Resource has been disposed. Disposed Resources are not to be used and are treated the same as a null value by most methods.
(Inherited from Resource.)
Fragment
[GET / SET] The FragmentShader that is used by this ShaderProgram.
FullName
[GET] The full name of the Resource, including its path but not its file extension
(Inherited from Resource.)
IsDefaultContent
[GET] Returns whether the Resource is part of Duality's embedded default content.
(Inherited from Resource.)
IsRuntimeResource
[GET] Returns whether the Resource has been generated at runtime and cannot be retrieved via content path.
(Inherited from Resource.)
Minimal
A minimal ShaderProgram, using a Minimal VertexShader and a Minimal FragmentShader.
Name
[GET] The name of the Resource.
(Inherited from Resource.)
Path
[GET] The path where this Resource has been originally loaded from or was first saved to. It is also the path under which this Resource is registered at the ContentProvider.
(Inherited from Resource.)
Picking
A ShaderProgram designed for picking operations. It uses a Minimal VertexShader and a Picking FragmentShader.
SmoothAnim
The SmoothAnim ShaderProgram, using a SmoothAnim VertexShader and a SmoothAnim FragmentShader. Some Renderers might react automatically to Materials using this ShaderProgram and provide a suitable vertex format.
SourcePath
[GET / SET] The path of the file from which the Resource has been originally imported or initialized. Setting this does not affect the Resource in any way.
(Inherited from Resource.)
UniformCount
[GET] Returns the number of uniform variables that have been declared.
VarInfo
[GET] Returns an array containing information about the variables that have been declared in shader source code.
Vertex
[GET / SET] The VertexShader that is used by this ShaderProgram.
